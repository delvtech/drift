# Community & Resources

## Contributing

We welcome contributions from the community! If you'd like to contribute, please read our contributing guidelines and open a pull request on [GitHub](https://github.com/delvtech/drift).

## Roadmap

Our roadmap is managed through GitHub Projects. You can view our public roadmap to see what we're working on and what's coming next.

## Changelog

For a detailed list of changes in each release, please see the [CHANGELOG.md](https://github.com/delvtech/drift/blob/main/packages/drift/CHANGELOG.md) file in the `packages/drift` directory.
