# Community & Resources

## Contributing

Contributions are welcome from the community! Here's how you can get involved:

- **Read the [Contributing
  Guidelines](https://github.com/delvtech/drift/blob/main/.github/CONTRIBUTING.md)**
  for setup instructions and development workflow
- **Open an [Issue](https://github.com/delvtech/drift/issues)** to report bugs
  or request features
- **Submit a [Pull Request](https://github.com/delvtech/drift/pulls)** with your
  improvements
- **Start a [Discussion](https://github.com/delvtech/drift/discussions)** to ask
  questions or share ideas

## Getting Help

- **GitHub Issues**: For bug reports and feature requests
- **GitHub Discussions**: For questions, support, and general discussion
- **Documentation**: Browse these docs for comprehensive guides and examples
