# API Reference

This section provides a high-level overview of the core Drift API. For detailed
information, please refer to the source files.

## Core

### `createDrift`

The main entry point for creating a Drift client.

- **Source:**
  [`packages/drift/src/client/Drift.ts`](https://github.com/delvtech/drift/blob/main/packages/drift/src/client/Drift.ts)

### `Drift`

The Drift client instance.

- **Source:**
  [`packages/drift/src/client/Drift.ts`](https://github.com/delvtech/drift/blob/main/packages/drift/src/client/Drift.ts)

### `contract`

A method on the Drift client to create a type-safe contract instance.

- **Source:**
  [`packages/drift/src/client/contract/Contract.ts`](https://github.com/delvtech/drift/blob/main/packages/drift/src/client/contract/Contract.ts)
