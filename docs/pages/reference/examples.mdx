# Examples

The Drift repository includes a variety of examples to help you get started.

## Example Applications

The full source code for our example applications can be found in the
[`examples/`](https://github.com/delvtech/drift/tree/main/examples) directory of
the repository.

### Node.js

Location:
[`examples/node.js/`](https://github.com/delvtech/drift/tree/main/examples/node.js)

This example demonstrates how to use Drift in a Node.js environment for
scripting and backend tasks. It includes examples of:

- Reading and writing to contracts
- Using multicall
- Bytecode calls
- Testing with mocks

### Web

Location:
[`examples/web/`](https://github.com/delvtech/drift/tree/main/examples/web)

This example shows how to integrate Drift into a web application using Vite and
React. It demonstrates how to use Drift for:

- Fetching data and populating the UI
- Sending transactions
- Integrating with React hooks

## Projects Using Drift

- [Doppler SDK](https://github.com/whetstoneresearch/doppler-sdk) - A TypeScript
  SDK for interacting with the Doppler protocol, a liquidity bootstrapping
  system built on Uniswap V4.
- [Flaunch SDK](https://github.com/flayerlabs/flaunch-sdk) - A TypeScript SDK
  for seamless interaction with the Flaunch protocol, an innovative platform
